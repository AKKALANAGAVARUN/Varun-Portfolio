- React Icons

## 💻 Local Development

1. Clone the repository
```bash
git clone https://github.com/AKKALANAGAVARUN/Varun-Portfolio.git
```

2. Install dependencies
```bash
cd Varun-Portfolio
npm install
```

3. Start development server
```bash
```
npm run dev
